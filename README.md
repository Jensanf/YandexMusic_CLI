# YandexMusic_CLI
Console YandexMusic player for likes_tracks. 
It used: 
        1. Python 3.7.5
        2. MOC (Music On Console) 2.6
        3. Ubuntu 19.10

Command to start from console:
< python3 main.py 10 & >  
Where <10> is a number of track, 
The <&> - makes the command run in the background.
